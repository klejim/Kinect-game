Description détaillée

Classe de base utilisée pour déterminer quels squelettes traquer lorsque en mode manuel.

Cette classe n'a qu'une unique méthode : choseTrackedSkeletons(), qui prend en argument une trame de données sur les squelettes et doit renvoyer un tableau de deux identifiants qui seront ceux que la kinect traquera. La classe étant purement virtuelle, il est impératif d'implémenter une classe en héritant pour pouvoir utiliser la sélection manuelle d'utilisateurs à traquer.
