§ KInitializeFlags

enum KtI::KInitializeFlags

Des flags utilisés pour initialiser la Kinect (voir NUI Constants ).

Par commodité, les flags INITIALIZE_AUDIO, INITIALIZE_SKELETON, INITIALIZE_DEPTH et INITIALIZE_COLOR sont passés tels quels à NuiInitialize, ils ont donc la valeur définie par NuiApi.h. La valeur des autres flags est analysée pour appeler les fonctions concernées avec les paramètres correspondant.

Valeurs énumérées
INITIALIZE_COLOR 

Utilise la détection d'images couleur.

INITIALIZE_SKELETON 

Utilise la détection de squelettes.

INITIALIZE_DEPTH 

Utilise la détection détection de frame de profondeurs.

INITIALIZE_AUDIO 

Détecte l'émission de son.

INITIALIZE_ALL 

Utilise tous les flux possibles.

SKELETON_SEATED_SUPPORT 

Traque les squelettes en seated mode (sans le bas du corps).

SKELETON_NEAR_MODE 

Traque en mode proximité (de 40 cm à 3 m).
